# Todo & Shopping List App - versione 1.0
Ecco il mio primo progetto completo per gestire le attività!
Questa applicazione ti permette di gestire separatamente una lista di cose da fare (**ToDo**) e una lista della spesa (**Shopping**), con un'interfaccia facile da usare e ottimizzata per ogni dispositivo.

---

## Funzionalità principali
* **Doppia lista**: hai una lista per le cose da fare e un'altra per la spesa, così puoi tenere tutto organizzato.
* **Dati sempre disponibili**: grazie all'uso della memoria locale del browser (`localStorage`), le tue task non si perderanno mai, anche se chiudi il browser o ricarichi la pagina.
* **Modifica facile**: puoi cambiare il testo delle task in tempo reale con dei prompt interattivi.
* **Feedback visivo**: le task completate vengono evidenziate, barrate e spostate automaticamente in un dropdown menu per le task completate, così puoi vedere subito cosa hai già fatto.
* **Design per tutti i dispositivi**: l'interfaccia è ottimizzata per funzionare bene sia sui computer che sui telefoni, grazie a delle regole CSS speciali.

## Tecnologie utilizzate
* **HTML5**: per creare una struttura semantica facile da capire e accessibile a tutti.
* **CSS3**: per un design moderno e facile da gestire, basato su flexbox per far funzionare bene l'app su tutti gli schermi, variabili CSS per una facile manutenzione e Media Queries per il responsive web design.
* **JavaScript (ES6)**: per gestire tutta la logica dell'applicazione, senza bisogno di librerie esterne.

## Come utilizzare il Progetto
1. Scarica tutti i file del progetto (HTML, CSS, JS).
2. Assicurati che tutti i file siano nella stessa cartella.
3. Apri il file `index.html` con un browser moderno.
4. Inizia ad aggiungere le tue task!
